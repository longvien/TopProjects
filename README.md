1. employee_payroll.py (Employee Payroll System)
Description:
A comprehensive payroll management program for different types of employees: salaried, hourly, and commission-based.
The program calculates weekly pay, taking into account multi-level inheritance and commission calculations.

2. vocabularyLearningProgram.py (Vocabulary Learning Program)
Description:
An interactive bilingual flashcard system that tests users on vocabulary randomly.
The program allows users to input words in two languages and quizzes them until they choose to stop.

3. automatic_calculator_class.py (Automatic Calculator Using Classes)
Description:
A calculator program that performs addition, subtraction, multiplication, and division using OOP.
Users input the operation symbol, and the program executes the correct calculation.

4. temperature_converter.py (Temperature Converter)
Description:
A program that converts temperatures between Fahrenheit and Celsius, based on user input.
The program handles multiple input formats (like °F to °C or F to C) and outputs results clearly.

5. automatic_calculator.py (Procedural Version)
Description:
A basic calculator program that performs addition, subtraction, multiplication, and division based on user input.
Users select the operation symbol, enter two numbers, and receive the result instantly.
